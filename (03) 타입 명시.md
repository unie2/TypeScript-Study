## 1. 타입 명시
  - 변수를 선언할 때, 변수 값의 타입을 명시함으로써 변수 값의 데이터 타입을 지정하는 것이다.
```javascript
// 변수 이름 뒤에 콜론(:)을 찍고 변수의 데이터 타입을 지정
let x : string = '나는 영원한 문자열'; 

// 지정된 데이터 타입 값만이 변수에 들어갈 수 있다.
```
